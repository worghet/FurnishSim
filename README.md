# FurnishSim

## about
FurnishSim (December 2023)  furniture business simulator. Manage your money, increase your popularity, deal with unexpected scenarios, and get rich!!

## tech stack
- java (swing ui)
- object oriented programming

## collaborators
This was a small group project, consisting of only 2 people.
- Me: I did most of the programming, including the tick system, AI, business asset manager, etc.
- @Creative-Name1 worked on the art, helped with coming up with concepts.

## skills learned
- working in a team.
- managing deadlines and team expectations.
- Applying OOP principles in a project.
